# Infrastructure Monitoring: Pavement Crack Detection & Analysis via YOLOv8

## Project Overview
This repository contains the methodology framework and automated computer vision pipeline developed for high-precision infrastructure health monitoring. The system integrates real-time object detection with analytical forecasting to monitor structural deterioration trends.


## Technical Highlights
* Developed a comprehensive data collection framework tailored for predictive deterioration analysis.
* Implemented advanced object detection using **YOLOv8** for precise, real-time crack localization.
* Integrated statistical and analytical forecasting models to predict long-term structural deformation.
